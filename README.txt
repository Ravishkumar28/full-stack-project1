# Menstrual Reviews Website

This project includes:
- React + Vite frontend
- Firebase Authentication (email login)
- Firestore Database to store stories
- "Understanding Menstruation" info panel
- User submission form
- Admin dashboard (email-filtered)
- All 23 real stories grouped by state and language

To run the project:
1. Install Node.js
2. Run `npm install`
3. Add your Firebase config to a `.env` file
4. Run `npm run dev`
